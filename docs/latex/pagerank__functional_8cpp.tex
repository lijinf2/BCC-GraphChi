\char`\"{}\-Functional\char`\"{} version of pagerank, which is quite a bit more efficient, because it does not construct the vertex-\/objects but directly processes the edges.

This program can be run either in the semi-\/synchronous mode (faster, but less clearly defined semantics), or synchronously. Synchronous version needs double the amount of I/\-O because it needs to store both previous and current values. Use command line parameter mode with semisync or sync.
