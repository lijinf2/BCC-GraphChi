Alternative \char`\"{}functional\char`\"{} A\-P\-I for Graph\-Chi. The A\-P\-I is implemented as a layer on top of the standard A\-P\-I, but uses a specialized engine \char`\"{}functional\-\_\-engine\char`\"{}, which processes the graph data in different order. Namely, it first loads in-\/edges, then executes updates, and finally writes new values (broadcasts) to out-\/edges.
